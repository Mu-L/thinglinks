## getDictLabel方法逻辑
1. 根据字典类型查找字典项列表
2. 在字典项列表中，使用字典项标识进行匹配，能匹配上的，
   2.1 优先取字典项中的 name 字段
   2.2 若配置了 i18nJson， 则根据当前语言环境，取i18nJson中的值
3. 若字典项列表中不存在 value 值， 则返回 defaultValue 默认值

## 字典使用 注意事项
1. 前端使用的所有字典，都应该先在后端进行配置，然后在将字典类型维护在DictEnum类中，否则前端无法使用useDictStore进行字典回显
2. 后端使用echoMap回显时，需要确保是同步线程，且前端请求头中需要封装Locale参数
