## ACL 权限检查服务

### 设计概述

一个基于双路径检查的MQTT权限控制系统，优先检查客户端本地元数据中的ACL规则（快速路径），未命中时回退到HTTP API检查（慢速路径）。

### 核心设计

```mermaid
graph TD
    A[客户端请求] --> B{元数据中有ACL规则?}
    B -->|是| C[执行本地规则匹配]
    B -->|否| D[调用ACL检查API]
    C --> E{规则匹配成功?}
    E -->|是| F[允许访问]
    E -->|否| G[拒绝访问]
    D --> H{API返回200?}
    H -->|是| F
    H -->|否| G
